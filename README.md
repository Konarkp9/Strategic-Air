# Strategic Air Analysis Dashboard

Interactive dashboard analyzing 7 quarters of airline simulation data with reverse-engineered algorithm mechanics and strategic recommendations.

## 🚀 View Live Dashboard

**[View Dashboard](https://konarkp9.github.io/Strategic-Air/strategic_air_analysis.html)**

## 📊 Features

- **KPI Metrics**: Stock price, profit, quality, reliability tracking
- **Interactive Charts**: Stock performance, profit trajectory, cost analysis
- **Algorithm Analysis**: Reverse-engineered demand, revenue, and cost models
- **Strategy Playbook**: Three proven strategies for next round
- **Competitor Intelligence**: Analysis of 6 competing airlines

## 📁 Contents

- `strategic_air_analysis.html` - Complete interactive dashboard (open in browser)

## 🎯 Key Insights

- Quality investment is the #1 profit lever (reduces refunds by ~50%)
- Cargo service adds $400-500K/quarter revenue
- Marketing has diminishing returns past $15-20K
- Conservative fleet management outperforms aggressive expansion
